Help

Flashcarder is an app that allows you to upload custom "decks" of flashcards with words and definitions.

To upload a deck, click "Upload" or drag and drop a file.

You can download a few premade examples here.

To make your own, the most surefire method is to use the editor, which allows you to modify the cards live. You can also choose to upload a json file. An example format can be found here.

You can also upload a text file (the extension doesn't matter). The file should consist of entries, with an optional set of special keys _title, _source, _sourceURL. An example format can be found here. The editor's "raw items" box works the same as an uploaded text file.

Note that the main delimiter can be a tab, semicolon, colon, or pipe. Repetitions of the main delimiter will be ignored. Currently, a comma is not supported due to a design oversight whereby the comma was used for multiple answers for a given entry. However, you can override this by including an optional line at the top of the file: SEP=X where X is the delimiter you want, following the Excel standard for CSV files.

You can also share your deck by copying a unique URL that encodes all the information. Since URLs have limited length, large decks (a few hundred words or more) may not work. Alternatively, you can share a file generated by the program and have them upload that. These files can be of any size.
